# Bandit Level 0
Goal: login to the game using ssh

## Concept learned: 
connect and login to a host with specific port by using username and password
## Resource for help: 
ssh on wikipedia, Linux Journey on LabEx

## Command used
ssh -p 2220 bandit0@bandit.labs.overthewire.org
## Result
first login!! shows us the welcome page text and some rules & regulations as well as tips.

## syntax: 
ssh -p port_number username@hostname
