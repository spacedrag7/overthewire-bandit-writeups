# Bandit Level 0 to 1
Goal:To find the password for logging in to the next level which is stored in a file called readme.

# Concept learned: 
navigate through the system by listing files and reading the content of a file.
# Resource for help: 
Linux Journey on LabEx

# Commands used
ls          //for listing directories or files and folders

cat readme  //concatenates and shows the content of a file

# Result
The ls command shows the content of the current directory, which only shows us the 'readme' file. The cat command displays the content of the readme file, which in this case gives us the password for logging in to the next level.

# syntax: 
# no.1
ls [options] [file/directory]

# no.2
cat [options] [filename]
